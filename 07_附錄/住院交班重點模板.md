# 住院交班重點模板

## 一句話重點

- 好的急診住院交班不是把所有資訊重講一次，而是清楚交代「病人現在有多危險、我們已做什麼、接下來最怕什麼」。

## 建議交班骨架

### 1. 病人是誰

- 年齡 / 性別
- 主要主訴
- 重要共病與基線功能

### 2. 為什麼要住院

- 最可能診斷
- 目前最需要監測或治療的問題
- 是否仍在 evolving disease course

### 3. 現在有多穩定

- 生命徵象趨勢
- 氧氣需求
- 疼痛、意識、尿量、灌流等重點
- 是否曾低血壓、低氧、心律異常或需要升級支持

### 4. 已做了什麼

- 已完成的重要檢查
- 已給的藥物、液體、血品、抗生素
- 已做的程序
- 已會診哪些科別

### 5. 還在等什麼

- Pending labs / cultures / imaging
- 待床邊 reassessment 的重點
- 是否有明確 disposition 轉折點

### 6. 最怕什麼

- 可能惡化的方向
- 需要立即回報的 red flags
- 若變差時的 next step

## 一句話版交班模板

- `這是 ___ 歲 ___，因 ___ 住院，目前最像 ___，剛剛已做 ___，現在還在擔心 ___，若出現 ___ 請優先 ___。`

## 高風險病人加強版

- 休克 / 敗血症：目前 MAP、乳酸、尿量、升壓藥與感染源控制進度
- 呼吸衰竭：氧氣模式、最近血氣、是否可能插管
- GI bleed：輸血量、最近 Hb、是否持續出血、內視鏡計畫
- 神經病人：基線神經狀態、最近變化、影像重點、是否已有專科評估

## 常見交班錯誤

- 只交 diagnosis，不交病情嚴重度
- 沒說 pending item
- 沒說最擔心的惡化方向
- 把正常值講很多，卻漏掉真正異常趨勢
